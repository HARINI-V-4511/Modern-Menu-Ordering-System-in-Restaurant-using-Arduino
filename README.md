 Project Overview

The **Modern Menu Ordering System in Restaurant using Arduino** is a smart automation project designed to replace traditional paper-based restaurant ordering methods.
Customers can view menu items on a **TFT touchscreen**, select their desired food, and send the order directly to the kitchen.

The order is transmitted wirelessly using an **RF transmitter and receiver module**, and the kitchen unit displays the order on an **LCD screen** while a **buzzer alerts staff**.

This system improves order accuracy, reduces waiting time, and enhances the dining experience.

 Components Used

* Arduino UNO
* 2.4" TFT Touchscreen Display
* FS1000A RF Transmitter
* XY-MK-5V RF Receiver
* 16×2 LCD Display with I2C Module
* Buzzer
* Jumper Wires
* Breadboard
* Power Supply

 System Working

1. Customer browses the menu on the TFT touchscreen.
2. Customer selects the desired food item.
3. Arduino processes the input.
4. The RF transmitter sends the order wirelessly.
5. The kitchen unit receives the order using RF receiver.
6. The order is displayed on the LCD screen.
7. Buzzer alerts the kitchen staff about the new order.
